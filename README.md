## Installation

```bash
pip install -U ckan-client-tacc
```

or install with `Poetry`

```bash
poetry add ckan-client-tacc
```

Then you can run

```bash
ckan-client-tacc --help
```

or with `Poetry`:

```bash
poetry run ckan-client-tacc --help
```
